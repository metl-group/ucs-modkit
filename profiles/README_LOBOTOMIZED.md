# UCS Modkit (Lobotomized Offline Build)

This package is a restricted offline distribution profile.

Behavior goals:
- No automatic dependency downloads.
- No automatic updates.
- No runtime network connections by design.
- Local file operations only.

Included binaries:
- `ucs_modkit_gui.exe`
- `ucs_modkit_cli.exe`

Notes:
- Loader installation uses bundled local files.
- Developer build actions are intentionally disabled in this profile.
